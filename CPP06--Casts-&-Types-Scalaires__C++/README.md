## <p align="center"> CPP06 - Casts et types en C++ </p>

### Aperçu

Le module **CPP06** se concentre sur les **conversions de types** en C++, en particulier l'utilisation des **casts explicites** (`static_cast`, `dynamic_cast`, `const_cast`, `reinterpret_cast`). Il explore également la conversion d’une chaîne de caractères vers différents types natifs (char, int, float, double).

##

### Compétences acquises

- **Conversion de types** : comprendre les comportements des conversions implicites vs explicites
- **Utilisation des 4 cast operators C++** :
  - `static_cast` : conversion sûre à la compilation
  - `dynamic_cast` : conversion entre types polymorphiques avec vérification à l'exécution
  - `const_cast` : ajout/suppression du `const`
  - `reinterpret_cast` : cast brut (dangereux)
- **Conversion de chaînes (`std::string`) en types fondamentaux**
- **Affichage et gestion des cas limites (impossible, overflow, etc.)**

##

### Exercices principaux

- **ScalarConverter (ex00)** :
  - Convertit une string vers `char`, `int`, `float`, `double`
  - Détection et affichage des cas spéciaux (`nan`, `+inf`, `-inf`)
  - Utilisation de `std::isprint`, `std::stoi`, `std::stof`, etc.
  - Résultat formaté avec gestion d'erreurs

##

### Pourquoi ce module est pertinent

Ce module vous apprend à **manipuler la représentation des types en profondeur**, à détecter les **erreurs de conversion**, et à **maîtriser les castings spécifiques au C++**, fondamentaux dans les systèmes fortement typés, les applications bas-niveau, ou le polymorphisme.
